# 📦 쇼핑몰 어드민 (Console App)

> **Java Collection Framework 기반 상품 관리 시스템**  
> 객체지향 설계를 바탕으로 상품 데이터를 콘솔에서 직접 관리하는 어드민 프로그램입니다.

---

## 📝 프로젝트 소개

### **"OOP 원칙을 직접 적용한 첫 번째 Java 프로젝트"**

**쇼핑몰 어드민 Console App**은 Java의 핵심 개념인 객체지향 프로그래밍을 실전 적용하여  
상품 등록·조회·수정·삭제(CRUD)를 콘솔 환경에서 처리하는 어드민 관리 시스템입니다.

---

## 🎯 제작 배경

- Java 문법 학습을 넘어, 실제 비즈니스 로직을 OOP 구조로 설계하는 경험이 필요함
- 단순 출력 코드에서 벗어나 데이터 흐름과 클래스 설계를 직접 고민하고 구현

---

## 🛠 기술 스택

- **Language**: Java
- **Database**: Oracle DB
- **IDE**: Eclipse

---

## 📁 프로젝트 구조

```
Console-based-Java-Project/
 ├─ src/
 │   ├─ model/       # 상품 데이터 객체 (Product DTO)
 │   ├─ dao/         # 데이터 접근 로직 (DB 연동)
 │   ├─ service/     # 비즈니스 로직 처리
 │   ├─ view/        # 콘솔 UI 출력 및 입력 처리
 │   └─ main/        # 프로그램 진입점
 ├─ bin/             # 컴파일된 클래스 파일
 └─ README.md
```

---

## 🧩 주요 기능
 
### 📦 상품 관리 (CRUD)
- 🛍 **상품 등록**: 상품명, 가격, 수량, 카테고리 입력 후 DB 저장
- 🔍 **상품 조회**: 전체 목록 조회 및 상품명 기반 검색
- ✏️ **상품 수정**: 상품ID 기반 정보 업데이트
- 🗑 **상품 삭제**: 선택 상품 삭제 처리
 
### 👤 회원 관리
- 회원 가입/수정/삭제
- 회원 정보 조회 (전체 회원 리스트)
- 로그인 인증 검증
 
### 🎁 할인 관리
- 상품별 할인율 설정
- 할인 정보 수정/삭제
- 할인율 적용된 최종 판매가 자동 계산
 
### 💬 리뷰 관리
- 상품별 리뷰 작성/수정/삭제
- 리뷰 목록 조회
 
---

## 🔄 User Flow
1. 프로그램 실행 → 메인 메뉴 출력
2. 메뉴 번호 선택 (1.회원관리 / 2.상품관리 / 3.리뷰관리 / 4.할인관리)
3. 선택한 모듈에서 추가 메뉴 선택
4. 입력값 처리 → DB 작업 실행
5. 결과 출력 → 메뉴로 복귀

 
---
 
### ✨ 학습 성과
- ✅ Java 객체지향 설계의 기초 이해 (캡슐화, 상속, 다형성)
- ✅ DAO/DTO 디자인 패턴으로 계층 분리
- ✅ Oracle DB JDBC 연동 및 CRUD 구현
- ✅ PreparedStatement로 SQL 인젝션 방지
- ✅ Collection Framework 활용 (ArrayList, HashMap)

- 모듈별로 분산되어 있어 통합 관리의 복잡성

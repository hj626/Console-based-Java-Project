# 📦 쇼핑몰 어드민 (Console App)

> **Java Collection Framework 기반 상품 관리 시스템**  
> 객체지향 설계를 바탕으로 상품 데이터를 콘솔에서 직접 관리하는 어드민 프로그램입니다.

---

## 📝 프로젝트 소개

### **"OOP 원칙을 직접 적용한 첫 번째 Java 프로젝트"**

**쇼핑몰 어드민 Console App**은 Java의 핵심 개념인 객체지향 프로그래밍을 실전 적용하여  
상품 등록·조회·수정·삭제(CRUD)를 콘솔 환경에서 처리하는 어드민 관리 시스템입니다.

---

## 🎯 제작 배경

- Java 문법 학습을 넘어, 실제 비즈니스 로직을 OOP 구조로 설계하는 경험이 필요함
- 단순 출력 코드에서 벗어나 데이터 흐름과 클래스 설계를 직접 고민하고 구현

---

## 🛠 기술 스택

- **Language**: Java
- **Database**: Oracle DB
- **IDE**: Eclipse

---

## 📁 프로젝트 구조

```
Console-based-Java-Project/
 ├─ src/
 │   ├─ model/       # 상품 데이터 객체 (Product DTO)
 │   ├─ dao/         # 데이터 접근 로직 (DB 연동)
 │   ├─ service/     # 비즈니스 로직 처리
 │   ├─ view/        # 콘솔 UI 출력 및 입력 처리
 │   └─ main/        # 프로그램 진입점
 ├─ bin/             # 컴파일된 클래스 파일
 └─ README.md
```

---

## 🧩 주요 기능

- 🛍 **상품 등록**: 상품명, 가격, 수량, 카테고리 정보 입력 및 저장
- 🔍 **상품 조회**: 전체 목록 조회 및 상품명 기반 검색
- ✏️ **상품 수정**: 상품 정보 업데이트
- 🗑 **상품 삭제**: 선택 상품 삭제 처리
- 📋 **메뉴 기반 UI**: 번호 입력 방식의 콘솔 메뉴 인터페이스

---

## 🔄 User Flow

1. 프로그램 실행 → 메뉴 출력
2. 메뉴 번호 선택 (등록 / 조회 / 수정 / 삭제)
3. 입력값 처리 → 비즈니스 로직 실행
4. 결과 출력 → 메뉴로 복귀

---

## 🌱 프로젝트 회고

- `else-if` 중첩 구조에서 문자열 인덱스 활용으로 코드 단순화 경험
- Collection Framework(ArrayList, HashMap)를 활용한 데이터 관리 흐름 체득
- 클래스 분리(DTO / DAO / Service / View)의 필요성을 직접 느끼며 MVC 구조의 기초 이해
- 단순히 "돌아가는 코드"가 아닌 "읽히는 코드"를 고민하는 계기가 된 프로젝트
